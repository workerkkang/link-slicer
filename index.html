사용법 요약
① 파일 선택으로 이미지 업로드
② 이미지 위 클릭 → 분할선 추가
③ Alt+클릭 → 해당 분할선 삭제
④ 각 조각 링크(URL) 입력
⑤ 코드 생성 → HTML 생성
⑥ 코드 복사 → 원하는 곳에 붙여넣기

이미지 분할 링크 생성기

이미지 위를 클릭해 가로 분할선을 추가하고, 각 조각에 개별 링크를 넣어 HTML을 생성하세요.

1이미지 업로드
• 클릭: 분할선 추가 • Alt/Option+클릭: 삭제 • 드래그: 이동
2각 조각 링크 입력
3코드 생성 & 미리보기
이 코드는 뉴스레터/블로그에 그대로 붙여넣을 수 있습니다. 일부 이메일 클라이언트에서는 data URL이 차단될 수 있으니 필요 시 이미지 호스팅을 변경하세요.
미리보기
팁
• 분할선은 원본 해상도 기준으로 저장됩니다.
• “코드 생성” 시 각 조각을 canvas로 잘라 <img src="data:...">로 만듭니다.
• 이메일 호환성 이슈 시, 생성 이미지를 서버에 업로드하고 src를 URL로 교체하세요.
